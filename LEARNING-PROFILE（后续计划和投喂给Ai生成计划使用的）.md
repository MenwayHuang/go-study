### 2.2 已掌握知识点

#### 🔵 高并发 (High Concurrency)

| 知识点 | 来源 | 掌握程度 | 核心理解 |
|--------|------|----------|----------|
| goroutine/channel | ch01 | ✅ | CSP模型，通过通信共享内存 |
| sync原语 | ch01 | ✅ | Mutex/RWMutex/WaitGroup/Once |
| context | ch01 | ✅ | 超时控制、取消传播、生命周期管理 |
| Worker Pool | ch02 | ✅ | 控制并发度，避免无限goroutine |
| Fan-out/Fan-in | ch02 | ✅ | 并行处理后汇总结果 |
| Pipeline | ch02 | ✅ | 多阶段流水线处理 |
| Rate Limiter | ch02 | ✅ | 令牌桶/滑动窗口/漏桶原理 |
| HTTP超时配置 | ch03 | ✅ | ReadTimeout/WriteTimeout等参数 |
| 消息队列削峰 | ch04 | ✅ | 异步解耦，平滑流量 |
| 缓存三大问题 | ch05 | ✅ | 穿透/雪崩/击穿的原理和解决方案 |
| SingleFlight | ch05 | ✅ | 相同key并发请求只执行一次 |

#### 🟢 高可用 (High Availability)

| 知识点 | 来源 | 掌握程度 | 核心理解 |
|--------|------|----------|----------|
| 优雅关闭 | ch03 | ✅ | Shutdown不丢失处理中的请求 |
| 中间件链 | ch03 | ✅ | 洋葱模型，横切关注点 |
| 服务注册发现 | ch06 | ✅ | 动态寻址，不硬编码IP |
| 负载均衡 | ch06 | ✅ | 轮询/加权/一致性哈希 |
| 熔断器 | ch06 | ✅ | 三态状态机，防止级联雪崩 |
| 重试策略 | ch07 | ✅ | 指数退避+抖动 |
| 优雅降级 | ch07 | ✅ | 核心保底，非核心可裁剪 |
| 幂等设计 | ch07 | ✅ | 重复执行不产生副作用 |
| 健康检查 | ch07 | ✅ | Liveness/Readiness |
